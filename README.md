# Strafcam

Strafcam is an Android application designed to turn an old or current Android phone into a smart dashboard camera. 

The core feature of this application is its ability to act as an automated reporting tool for High Occupancy Vehicle (HOV) / carpool lane violations in Seattle, WA.

## Planned Features
- **Dashcam Loop Recording:** Continuous video recording that automatically overwrites the oldest footage.
- **Computer Vision (HOV Detection):** 
  - Detection of the carpool lane lines (diamonds).
  - Detection of the number of occupants in the vehicle ahead or passing by.
  - License plate recognition (ALPR).
- **Automated Reporting:** 
  - Automatically triggers a call to the WSDOT HERO program (877-764-HERO) when a violation is confirmed.
  - Generates automated voice prompts or logs the details.
